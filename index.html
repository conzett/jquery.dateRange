jQuery.dateRange by conzett

A single field date range picker that doesn't require jQuery UI

jQuery.dateRange is a jQuery plugin that provides an interface for picking a range of dates for a single text field. The generated markup is clean and accessible with aria roles and lean markup. A simple default style is included (modeled after the iOS calendar).

Demo

Dependencies

There are NO additional dependencies. (Other date-pickers often require Datejs or jQuery UI)
